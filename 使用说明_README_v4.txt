俄罗斯旅行浏览器插件智能导入工具包 v4（设施识别增强）

改进点：
1. 插件读取前会尽量点击“显示全部/阅读更多/Показать все”，减少酒店设施被折叠导致的缺失。
2. 后端 /ai-extract 不再只取第一个“酒店服务”，而是对多个候选设施区块评分，避开顶部导航。
3. 增强 Ozon 酒店设施关键词：停车、宠物/禁宠、禁烟、餐厅、酒吧、冰箱、水壶、空调、Wi‑Fi、浴室/淋浴、吹风机、毛巾、拖鞋、浴袍、电视、电梯、洗衣、健身房、会议厅、安全/行李寄存、员工语言、无障碍等。

使用：
1. 先停止旧服务 Ctrl+C。
2. 运行：python travel_plugin_import_server_v4.py
3. 浏览器扩展程序页面重新加载 travel_page_capture_extension。
4. 打开 俄罗斯横穿旅行_插件接收版_v4_设施增强.html。
